# Sift: Simple Shopping List

Sift: Simple Shopping List is an iOS application available on the Apple App Store. It is a free, easy-to-use, and easy-to-read shopping list and recipe organizer designed to streamline everyday meal planning and grocery management.

## Features

* Intuitive shopping list management
* Clean and readable interface
* Simple recipe organization
* Fast item entry and categorization

## Availability

Sift is available for download on the Apple App Store.
